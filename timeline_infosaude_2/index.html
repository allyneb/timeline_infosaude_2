SOFTWARES E SISTEMAS OPERACIONAIS

Os mais conhecidos sistemas operacionais são: o Windows, OSx, Android, iOS e o Linux, cada um com suas várias versões. Esse é o papel do software Sistema operacional. Controlar tudo o que acontece dentro de um computador.

Outro software muito interessante é o driver. Você já deve saber como ele funciona, mas vamos relembrar. Quando um novo dispositivo chega ao computador o sistema operacional logo reconhece e tenta se comunicar com o dispositivo. Porém a CPU não consegue executar o comando, pois não entende as informações do dispositivo. Dessa forma, o Sistema operacional pede a CPU para procurar novos tradutores para os dispositivos. Esses tradutores são programas chamados Drivers. Quando a CPU encontra o driver específico, o sistema operacional consegue se comunicar perfeitamente com o novo dispositivo.

A CPU, então, começa a se comunicar com o dispositivo. E tudo fica feliz e funcionando. Mas existem outros tipos de programas. Existe um tipo de programa chamado de software para linha de produtos, que são os softwares mais comuns nos computadores. São as planilhas de texto, os editores de texto, leitores de música e muitos outros.

Outro tipo de software que está muito em desenvolvimento são os aplicativos para web. Eles apresentam uma característica muito interessante. Uma parte dos processamentos não é feita pela CPU do seu computador e sim de um outro computador ou servidor. Isto é, seu computador é o caminho para acessar o conteúdo que está disponível. Esse é o tipo de aplicativo conhecido como aplicativo para WEB, o qual só uma parte do trabalho é realizado pelo seu computador, mas o principal é a busca das informações na internet. Existem alguns outros tipos de programas, mas esses são os mais importantes. Como você viu, todos são software, mas cada um trabalha de uma forma diferente.
